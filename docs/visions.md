# NC-PoRe Vision

## Projektidee

NC-PoRe ist eine selbstgehostete Open-Source-Podcast-Produktionsplattform.

Das Ziel ist eine datenschutzfreundliche Alternative zu cloudbasierten
Podcast-Aufnahmesystemen.

## Grundprinzip

Audio wird lokal auf den Endgeräten der Teilnehmer aufgezeichnet.

Die Netzwerkverbindung darf nicht die Qualität der Aufnahme bestimmen.

## Werte

- Datenhoheit
- offene Standards
- FOSS
- Selbsthosting
- transparente Architektur
- professionelle Audioqualität
